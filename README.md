Requirement
===========
* ~ 20 GB free space

Example
=======
```
sudo mkdir /kernel-build
sudo chown USER:GROUP /kernel-build
cd /kernel-build
git clone https://github.com/plitc/custom_debian_kernel

cd /kernel-build/custom_debian_kernel
./custom_debian_kernel.sh
```
